兔兔桌寵 Windows 版 🐰
=====================

安裝步驟
--------
1. 把整個資料夾解壓縮到任意位置（所有 DLL 檔都要跟 shijima-qt.exe 放在一起）。
2. 雙擊 shijima-qt.exe 啟動。
   - 第一次開啟時 Windows 可能跳出「Windows 已保護您的電腦」：
     點「其他資訊」→「仍要執行」。
3. 開啟後會出現管理視窗，把資料夾裡的 usagi-shimeji.zip
   直接拖進管理視窗匯入。
4. 在清單裡雙擊兔兔就會生出一隻，想要幾隻就點幾次。

日常使用
--------
- 對兔兔按右鍵有選單：餵食、加水、生更多兔兔、🎉 下班狂奔等。
- 兔兔肚子餓或該喝水時，頭上會冒泡泡，也會發系統通知。
- 18:00 下班時間會提醒你，然後全場狂奔 20 秒慶祝。
- 要關掉：管理視窗選單裡選結束，或對兔兔右鍵選單操作。
