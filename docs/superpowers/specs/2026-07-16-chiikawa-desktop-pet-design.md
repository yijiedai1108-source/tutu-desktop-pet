# 吉伊卡哇桌面寵物 — 設計文件

- 日期：2026-07-16
- 狀態：已核可（口頭 OK），待實作
- 授權注意：吉伊卡哇為 nagano 版權角色，本素材包**僅供本人自用，不公開發佈**。

## 目標

在 macOS 桌面上跑一隻吉伊卡哇（本體，白色那隻）懸浮桌寵，具備：

1. 自主行為：亂走、發呆、坐下、睡覺（隨機切換）
2. 滑鼠互動：可點擊、可拖曳丟擲（拖著時手腳揮舞＋驚嚇表情）
3. 視窗互動：站在視窗頂端、視窗移動時掉落
4. 養成元素：本版僅含引擎內建餵食；心情值／不餵哭哭延後至第二階段

## 方案決策

- **選定：B+C 混合**——第一階段用現成引擎＋自製素材（本文件範圍），第二階段再補養成。
- 引擎：[Shijima](https://pixelomer.itch.io/shijima)（免費、GPLv3、原生 macOS 15+、相容 Shimeji-ee 素材包、免 Java）。
- 棄案紀錄（2026-07-16）：
  - itch.io「Desktop Chiikawa」：僅 Windows。
  - 現成同人素材包：查無吉伊卡哇本體 Shimeji（僅有兔兔 Usagi 的老 Java 版）。
  - 從零自製 app（純 C 案）：工程量大、視窗互動需自行實作，養成需求不急迫故延後。

## 架構

```
Shijima.app（引擎：物理、行為排程、視窗偵測）
   └── chiikawa-shimeji.zip（本專案產出）
         ├── img/  shime1.png ~ shime46.png（128×128 透明 PNG）
         └── conf/ actions.xml + behaviours.xml（沿用 Shijima 預設包，僅必要處微調）
```

- 行為腳本不從零寫，拿預設素材包 conf 起手；素材檔名對齊標準 Shimeji 編號。
- 工作量集中在美術：SVG 畫姿勢 → 批次轉 128×128 PNG。SVG 原始檔保留（`art/` 資料夾），日後調整五官或加角色（兔兔、小八）可重用。

## 素材規格

- 造型：白圓身、小耳朵、粉腮紅、豆豆眼，比例貼近原作。
- 46 張 = 十幾個基礎姿勢＋差分：站立、走路 2 幀、坐下、趴平、爬牆 2 幀、爬天花板 2 幀、被拖曳（手腳揮舞＋驚嚇）、掉落（淚眼）、彈地、睡覺等。
- 情緒彩蛋：掉落／被丟＝淚眼；坐下發呆差分＝捧臉。

## 驗收流程

1. **姿勢一覽表過稿**：先出一張全部姿勢的排版圖，使用者點頭才量產全套 PNG。
2. 打包 zip；使用者自 itch.io 下載 Shijima（免費）並匯入。
3. 實測檢查：走路流暢度、爬視窗破圖、拖曳表情、多寵物生成。

## 第二階段接點（本版不做）

- 養成（心情值、餵食紀錄、不餵哭哭）：選項為獨立選單列小 app 或 fork Shijima（GPLv3）加行為；sprite 全數沿用。

## 風險

- Shijima 相容性：若預設 conf 與自製圖有對不上的動作編號，以 Shijima 內建檢查器（inspector）除錯。
- 視窗互動在部分環境可能失效（itch.io 評論見於 Linux；macOS 待實測），失效時退回螢幕邊緣行走，不影響其餘功能。
